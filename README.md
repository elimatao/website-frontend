
# elia.doumerc.me
This is the source code of my personal website, which is available at [elia.doumerc.me](https://elia.doumerc.me).

## Development instructions
To run this project locally, you need to have [Node.js](https://nodejs.org) installed on your machine. Then, you can follow these steps:
1. Clone the repository:
```bash
git clone
```
2. Navigate to the project directory:
```bash
cd elia.doumerc.me
```
3. Install the dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
5. Open your browser and go to [http://localhost:3000](http://localhost:3000) to see the website.
